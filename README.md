<p>App created to teach people how work with React-Native. This step you will learn how navigate and open another pages in your application. To do this we use react-navigation who is a stack/component for manage our pages.</p>

## CREATE PROJECT
```sh
$ react-native init {project name}
```

Have two way to run app, one is open the project in Xcode (for ios) or Android Studio (for Android). Other way is open this page: https://facebook.github.io/react-native/docs/getting-started.html and look how configure the cli to use this command:

## RUN IOS
```sh
$ react-native run-ios
```
## RUN ANDROID
```sh
$ react-native run-android
```
